# Banana Pi BPI-M4 Zero with Allwinner H618 chip design with 2G RAM

## Specification
- Allwinner H618, Quad-core ARM Cortex™-A53 processor
- ARM Mali G31 GPU
- 2.4G/5G WIFI & Bluetooth 4.2
- 2G LPDDR4 RAM
- 8G eMMC flash memory
- 1x USB2.0 Type-C OTG, 5V power supply
- 1x USB2.0 Type-C
- 1x miniHDMI 2.0a
- 1x 24-pin FPC connector
- 1x USB2.0
- 1x 100Mbps Ethernet
